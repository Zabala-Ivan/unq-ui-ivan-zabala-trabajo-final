PREGUNTADOS


+CÓMO LEVANTAR EL PROYECTO


1- Clonar el repositorio 

git clone https://github.com/Zabala-Ivan/unq-ui-ivan-zabala-trabajo-final.git


2- Ingresar a la carpeta del juego

cd preguntados


3- Instalar las dependencias

npm install


4- Iniciar la pagina 

nmp start


+INSTRUCCIONES PARA JUGAR

1- Inicio: Hacer click en el boton "JUGAR" 

2- Selección de dificultad: Seleccionar haciendo click alguna de las dificultades disponibles, luego hacer click en el botón "INICIAR" para inciar el juego

3- Preguntas y respuestas: En cada una de las preguntas hay que elegir una de las 4 opciones que se crea es la correcta. Al seleccionar la respuesta va a tornarse de color ver en caso de ser correcta, o de color rojo  en caso contrario, luego automaticamente va a pasar a la siguiente pregunta

4- Fin del juego: cuando se terminen las preguntas vamos a poder ver la cantidad de respuestas correctas y tambien el boton "JUGAR DE NUEVO" que nos va a redirijir a la seleccion de dificultad para poder empezar una nueva partida
